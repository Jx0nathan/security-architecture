# 01 — 边界安全（Perimeter Security）
**对应 MAS TRM §9 — 网络边界控制**
当前状态：部分已实施（事故后持续加固中）

---

## 1. Cloudflare WAF 配置规范

**必须开启的配置项：**
```
安全等级：Medium（不允许设为 Low）
Bot 拦截模式：开启
浏览器完整性检查：开启
热链接保护：开启

WAF 托管规则：
→ Cloudflare OWASP 核心规则集：开启（灵敏度：Medium）
→ Cloudflare 托管规则集：开启
→ Cloudflare 凭据泄露检测：开启

限速规则（必须配置）：
→ 同一 IP 每分钟请求 /public/ 超过 100 次 → 触发验证
→ 同一 IP 每分钟请求 /auth/ 超过 50 次 → 封禁 1 小时
→ 同一 IP 每分钟请求 /user/public/v1/pre-login 超过 10 次 → 触发验证
→ 同一 IP 每分钟任意路径超过 200 次 → 封禁 24 小时

自定义拦截规则（必须配置）：
→ 拦截：URL 中包含 sleep( 或 ${ 或 #{ 或 .to_i（注入探测特征）
→ 拦截：URL 中包含 ..;/ 或 //internal（路径穿越攻击）
→ 拦截：User-Agent 匹配 python-urllib 或 python-requests（自动化工具特征）
→ 验证：敏感接口收到无标准浏览器请求头的请求
```

**IP 白名单变更管控策略：**
```
【重要】任何 WAF IP 白名单规则的变更必须满足：
→ 提交变更审批工单（CAB Ticket）
→ 双人授权（申请人 + 审批人不能是同一人）
→ 书面变更理由
→ 90 天自动过期（到期必须重新审批续期）
→ 变更操作记录进审计系统

【备注】此条规定是 2026 年 2 月事故的直接根因管控措施。
```

## 2. AWS WAF（OWASP 核心规则集）

**Web ACL 规则配置（按优先级排序）：**
```
规则 1：AWS-AWSManagedRulesCommonRuleSet        → 拦截 SQL 注入、XSS、RFI
规则 2：AWS-AWSManagedRulesKnownBadInputsRuleSet → 拦截 Log4j、SSRF 攻击
规则 3：AWS-AWSManagedRulesAdminProtectionRuleSet → 保护 /admin/ 路径
规则 4：AWS-AWSManagedRulesSQLiRuleSet           → 额外 SQL 注入防护
规则 5（自定义）：BlockInternalNamespace          → 拦截所有 /internal/ 路径访问
规则 6（自定义）：RateLimit-每IP每分钟100次限制

WAF 日志配置：
→ 所有 WAF 决策（放行 + 拦截）记录到 S3 + CloudWatch
→ 保留期：1 年
→ 告警条件：同一 IP 在 5 分钟内触发超过 10 次拦截 → 立即告警
```

**事故后必须添加的核心规则：**
```json
{
  "Name": "封锁公网访问内部API",
  "Priority": 1,
  "Statement": {
    "ByteMatchStatement": {
      "SearchString": "/internal/",
      "FieldToMatch": {"UriPath": {}},
      "TextTransformations": [{"Priority": 0, "Type": "LOWERCASE"}],
      "PositionalConstraint": "CONTAINS"
    }
  },
  "Action": {"Block": {}}
}
```

## 3. TLS 加密配置规范

**ALB 必须使用的 TLS 策略：**
```
安全策略：ELBSecurityPolicy-TLS13-1-2-2021-06
最低 TLS 版本：1.2
TLS 1.3：开启
禁用：SSLv3、TLS 1.0、TLS 1.1

允许的加密套件：
→ TLS_AES_128_GCM_SHA256（TLS 1.3）
→ TLS_AES_256_GCM_SHA384（TLS 1.3）
→ TLS_CHACHA20_POLY1305_SHA256（TLS 1.3）
→ ECDHE-RSA-AES128-GCM-SHA256（TLS 1.2）
→ ECDHE-RSA-AES256-GCM-SHA384（TLS 1.2）
```

**所有响应必须包含的安全响应头：**
```
Strict-Transport-Security: max-age=63072000; includeSubDomains; preload
X-Content-Type-Options: nosniff
X-Frame-Options: DENY
Content-Security-Policy: default-src 'self'; script-src 'self'
X-XSS-Protection: 1; mode=block
Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy: camera=(), microphone=(), geolocation=()
```
